      Hugo安装 

          winget install Hugo.Hugo.Extended

      创建Hugo站点 

          hugo new site my-hugo-site

      创建文章 

          hugo new posts/my-first-post.md

      打开文章，将文章draft状态改为false

      预览内容 

          hugo server

      生成静态文件 

          hugo

      然后上传到GitHub仓库

      Vercel上传

      1. 创建Vercel账号

      访问 Vercel官网 并注册一个账号。

      2. 创建新项目

      登录后，点击 “New Project” 按钮，选择 “Import Git Repository”。

      3. 连接GitHub仓库

      选择你的Hugo项目所在的GitHub仓库，并点击 “Import”。

      4. 配置项目设置

      在项目设置中，确保构建命令设置为 hugo

      5.设置环境变量

      设置环境变量 HUGO_VERSION 为你所使用的Hugo版本，例如 0.150.0。

      [!NOTE] 记得编辑 config.toml 文件 中的 baseURL 为你的Vercel域/域名。
